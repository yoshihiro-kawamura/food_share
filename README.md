## Taste share
自分のオリジナルなご飯を紹介していくアプリ

## 概要
一人暮らしをしている男性は食が毎日同じだったり、凝った料理は作れない、という人がたくさんいらっしゃるかと思います。
私自身も、晩ご飯の献立に悩み結局はカップラーメンや、冷凍食品に走ってしまうことが多々有ります。
自分を含めそんな人の悩みを解消したく、簡単で、美味しい料理をshareできるアプリを作りたいと思いました！
